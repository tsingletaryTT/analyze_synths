The story

From a library management problem to Blackhole silicon

The problem

A growing library of synthesizer pieces — long-form ambient tracks, generative drones, evolving textures — with no good way to understand what was in it. Standard tools give BPM and key. Not useful when the track has no tempo and the key is ambiguous.

The goal was a creative vocabulary: spacey, crystalline, droning, organic, oozy. Descriptors that mean something to a synthesizer composer.

v1 — Feature extraction

A librosa-based pipeline extracting 80+ features per track mapped through empirically-tuned thresholds to 176 descriptors: 9 core moods, 8 extended, 100 advanced, 59 character tags. K-means clustering, phase detection, and sequence recommendation on top.

It worked. The results felt right. The problem was speed on long files.

The insight

STFT is just three matrix multiplications. Given pre-computed static bases:

cos_proj = frame @ cos_basis
sin_proj = frame @ sin_basis
mag = sqrt(cos² + sin² + ε)
mel = mag @ mel_filter

That’s a kernel the hardware was built for.

TT-Lang kernel

The first hardware target was TT-Lang (ttnn), Tenstorrent’s Python-like compute language. Prototype ran. Then JAX PJRT arrived: compile the same matmuls with jax.jit, let the XLA backend lower to Blackhole Tensix cores. Cleaner, faster, portable.

The hardware journey

Four P300C Blackhole chips in a P150X4 mesh. Device reset between runs (stuck ethernet cores after a killed process). Single-process dispatch — multi-threading causes lock contention on chip init. JIT compiles once per kernel config, shape-padded to 1376 frames.

Full library (82 tracks)

Material from 1990s through present: analog synthesizers, drum machines, spoken word, generative pieces, ambient drones. 4 clusters by spectral similarity. 1,599 musical sections across 6.9h of audio. Every track sequenced by the recommended listening arc.

What’s next

Directions from here

Narrative generation

Phase-level prose descriptions beyond tag lists. A language model reading the energy arc and writing what it hears — not a transcript, a listener’s impression.

Cross-piece similarity

Cluster membership gives broad groupings. Next step: pairwise spectral distance for a proper similarity graph — “this piece sounds like that one because of X.”

Parallel dispatch

The single-process constraint on TT hardware is a workaround for JAX init lock contention. The right fix is a single shared device context with queue-based dispatch.

Expanded vocabulary

176 descriptors are a start. The advanced mood library has 100 more. Calibrating them against spoken word, 1990s drum machines, and acoustic instruments requires different thresholds.

Real-time analysis

The kernel already operates at 3,931x real-time. A streaming mode that reads from a DAW or audio interface and builds sections on the fly is architecturally feasible.

Export to DAW

Phase boundaries and mood tags as DAW markers. Import a 25-minute generative piece and get section markers automatically placed with creative labels.
